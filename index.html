Projects

Tech Tips daily

For my first-year final exam in frontend development, I created Tech Tips Daily, a frontend tips and tricks blog. Using a given API, I implemented features to fetch, post, edit, and delete blog posts. The site included a carousel of the 3 latest posts. I also implemented a login for the admin and the possibility for users to sign up through the API. Initially, progress was slow but picked up quickly. Designed in Figma with a sleek black, grey, and white theme, the project focused heavily on JavaScript, enhancing my skills significantly. The end result was highly satisfying.

Science Museum

For my first semester final exam, I created Science Museum, an online museum for children aged 7-14. The process was a long and educational journey. I learnt a lot about CSS and HTML through this project and the process went smooth and according to plan. Using Figma, I designed a vibrant and playful site, then implemented it using HTML and CSS. The design system was responsive, adapting to mobile, tablet, and PC screens with media queries. I used SourceTree for Git management and launched the site on Vercel. The result is an engaging, interactive platform that brings science to life for kids!

Frame Photo

For our Agency 1 class, we created Frame Photo, a website for wedding, Tinder, bourdoir, and headshot photography. The group consisted of me and 4 other classmates. The assignment emphasized planning and sprint execution over coding. We used sprints to manage our progress and met up every week for a meeting to ensure timely completion. We used figma for design, HTML, CSS and JavaScript. It was a great learning about how to work in a group since I had not done that before.
